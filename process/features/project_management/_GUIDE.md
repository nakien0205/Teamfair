# Project Management Feature Guide

This directory holds the implementation plans, backlog, and archives for the **Project Management** feature set.

## Scope
- **Workspace Selection:** Listing, creating, or joining workspace groups.
- **Invite Codes:** Generating and consuming invite codes (`IV-XXXXXX`) instead of raw UUIDs.
- **Onboarding requests:** Leader approval/rejection panel for applicants, pending state card.
- **Notifications:** Dynamic sidebar-navigable full-page notifications center.

## Target Directories
- `active/`: Active plans relating to Project Management
- `completed/`: Archived plans
- `backlog/`: Future features/backlog items
