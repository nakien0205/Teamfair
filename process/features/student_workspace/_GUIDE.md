# Student Workspace Feature Guide

This directory holds the implementation plans, backlog, and archives for the **Student Workspace** feature set.

## Scope
- **Dashboard & Layout:** StudentDashboard page with sub-components wrapped under StudentLayout.
- **Task Kanban:** Kanban board component with evidence uploads (stored on Supabase Storage).
- **Project Calendar:** Month/week calendars, event creation, and deadline sync.
- **Fairness & Analytics:** Contribution analytics dashboard, reporting peer members to lecturers.
- **AI Agent Sidebar:** OpenRouter-powered chat assistant.

## Target Directories
- `active/`: Active plans relating to Student Workspace
- `completed/`: Archived plans
- `backlog/`: Future features/backlog items
