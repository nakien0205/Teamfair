# Lecturer Workspace Feature Guide

This directory holds the implementation plans, backlog, and archives for the **Lecturer Workspace** feature set.

## Scope
- **Overview & Dashboard:** LecturerDashboard page rendering group tables, AI analytics, and overall metrics.
- **Class/Group Management:** LecturerGroupsPage class listing, projects, group scoring with pagination.
- **Rubrics & Grading:** Rubric upload (.xlsx/.csv), interactive editor, and split-screen grading panel.
- **Badges & Reviews:** Verified badges assignment and reviews publishing.

## Target Directories
- `active/`: Active plans relating to Lecturer Workspace
- `completed/`: Archived plans
- `backlog/`: Future features/backlog items
